# California Housing Data Description

The California Housing dataset was derived from the 1990 U.S. Census and first appeared in Pace & Barry (1997), "Sparse Spatial Autoregressions," Statistics and Probability Letters. It is distributed as a built-in dataset in scikit-learn. Each observation represents a census block group — the smallest geographical unit for which the Census Bureau publishes sample data, typically containing 600 to 3,000 residents. The target variable is the median house value for households within the block group, capped at $500,000.

The eight predictor features capture economic conditions (median household income), housing stock characteristics (median house age, average rooms and bedrooms per household), population density measures (block group population, average household occupancy), and geographic location (latitude and longitude). The dataset contains 20,640 observations with no missing values. Key analytical challenges include strong spatial autocorrelation (nearby block groups have similar prices), non-linear relationships between income and house value, and the target censoring at $500,000 which creates a ceiling effect for the most expensive areas. This is a well-established regression benchmark that tests a model's ability to capture both linear and non-linear feature effects alongside geographic clustering patterns.
