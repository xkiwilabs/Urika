# Text Sentiment Data Description

This synthetic dataset was created for testing Urika's natural language processing capabilities on a binary sentiment classification task. Each row contains a short product review (1-2 sentences) and a sentiment label (positive or negative). The reviews were generated from carefully designed sentence templates populated with varied adjectives, nouns, and adverbs to simulate realistic consumer language patterns. Positive reviews use language associated with satisfaction, quality, and recommendation; negative reviews reference defects, disappointment, returns, and poor customer service.

The dataset contains 200 reviews with a balanced class distribution (100 positive, 100 negative). Because the reviews were template-generated, they contain recognizable lexical patterns that should be learnable by standard text classification methods (bag-of-words, TF-IDF with logistic regression or SVM, etc.). The expected baseline accuracy is around 75-85% with simple feature extraction and a linear classifier. This dataset is intended to test Urika's ability to handle unstructured text data, apply appropriate preprocessing (tokenization, stopword removal, feature extraction), select suitable classification models, and evaluate performance using metrics appropriate for balanced binary classification (accuracy, precision, recall, F1).
