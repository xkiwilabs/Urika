# Image Classification Data Description

This synthetic dataset was created to test Urika's ability to handle image-based data sources, including file type detection, directory-structured dataset profiling, and basic image classification pipeline orchestration. The dataset contains 50 small RGB images (64x64 pixels) in PNG format, organized into three class subdirectories: cats (17 images), dogs (17 images), and birds (16 images). The images are not real photographs but synthetically generated patterns.

Each image consists of random pixel noise with a class-specific color tint applied: warm orange for cats, blue-purple for dogs, and green for birds. Geometric shapes (circles) are randomly overlaid for additional texture variety. The color tinting provides a trivially separable signal, meaning that even simple pixel-based features (average color channels) should achieve high classification accuracy (>90%). This design is intentional — the dataset tests Urika's data handling infrastructure rather than the difficulty of the classification problem itself. Expected analysis approaches include pixel-level feature extraction, color histogram analysis, or simple convolutional models, all of which should succeed given the clear color separation between classes.
