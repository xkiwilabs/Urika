*Urika: An Agentic Scientific Analysis & Modelling Platform for Behavioral and Health Sciences*

Short initial description:

Urika is a multi-agent system that investigates, models, and discovers analytical methods across behavioral and health sciences. Users define an investigation — a dataset, a question, and success criteria — and Urika's agents collaborate to explore the data, test competing approaches, and converge on optimized solutions. A Task Builder Agent scopes the investigation with the user and spawns a Task Agent equipped with a growing library of analytical tools. An Evaluation Agent critiques each run, searches literature, and suggests unexplored directions. A Tool Builder Agent constructs or extends capabilities on demand. Target domains span statistical modelling, machine learning, time series, neuroscience, cognitive neuroscience, linguistics, psychology, motor control, and behavioral data broadly.
